# Download
Step 1 - Add the JitPack repository to your root build.gradle file
```
allprojects {
  repositories {
    ...
    maven { url 'https://jitpack.io' }
  }
}

Step 2 - Add the dependencies:

dependencies {
  implementation 'com.github.rudsonlive:searchliveo:1.0.1'
}
```
